# University-campus-network-packet-tracer Project
Designed and implemented a University Campus Network in Cisco Packet Tracer featuring VLAN segmentation, inter-VLAN routing, RIP v2 dynamic routing, DHCP services, WAN serial connectivity, trunking, and multi-department network architecture. Demonstrates practical CCNA-level routing, switching, and troubleshooting skills.
# University Campus Network Design - Cisco Packet Tracer Project


## Features
- VLAN Segmentation
- Inter-VLAN Routing
- RIP v2 Dynamic Routing
- DHCP Configuration
- WAN Serial Connectivity
- Router-on-a-Stick
- Trunk & Access Port Configuration
- Multi-Switch Campus Topology
